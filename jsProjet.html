Champ de mines

Présentation du projet

Ce projet a été réalisé durant les séances de TD.

Ce jeu consiste à atteindre le trésor en évitant les mines, sinon la partie est perdue. Vous pouvez détecter les mines si vous êtes sur une case adjacente, ce qui vous aidera grandement à gagner. Un squelette de code nous a été donné pour aider mais sinon le déplacement, les messages et l'affichage de mines en cas de défaite ont été faits par moi.
